[![Status](https://img.shields.io/badge/status-in%20development-orange.svg)]()

## 项目简介

电子档案归档章印制工具，用于批量处理PDF档案文件，自动加盖符合T22-2015标准的归档章，并添加自定义页码。支持多种档案类型（文书、科技、会计、声像等），具备智能模板匹配、字段自动解析、断点续传等功能。

## 核心功能

- ✅ **批量PDF处理**：递归扫描多层文件夹，批量加盖归档章
- ✅ **T22-2015标准归档章**：符合档案行业规范
- ✅ **智能字段解析**：从文件路径和文件名自动提取全宗号、年度、件号等信息
- ✅ **档案类型模板系统**：支持多套规则并存，自动匹配档案类型
- ✅ **页码全量自定义**：支持范围、格式、位置、外观的完整自定义
- ✅ **保留双层PDF结构**：非破坏性编辑，保留OCR文本层
- ✅ **断点续传**：大批量处理中断后可恢复
- ✅ **配置方案管理**：保存和加载不同场景的配置方案

## 技术栈

- **前端**：React + TypeScript + Vite
- **后端**：Tauri v2 + Rust
- **数据库**：SQLite (sqlx)
- **PDF处理**：lopdf (保留双层PDF结构) + pdfium-render (预览辅助)
